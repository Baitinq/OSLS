# OSLS (Overly Simple Launch Simulator)

## Running
```
$ nix develop
$ python src/main.py
```

## Controls
```
x -> toggle engine
z -> stage separation
l/r arrows -> engine gimbal left/right
u/d arrows -> engine thrust up/down
+/- -> speedup/speeddown time
shift + (+/-) -> speedup/speedown physics
space -> pause
```

```
Check inside shell.nix :)
```
